# Projeto Triflix

Este projeto foi desenvolvido como parte da disciplina Desenvolvimento Front-End, utilizando o framework Next.js.
O principal objetivo é introduzir os alunos às ferramentas e práticas essenciais do desenvolvimento de interfaces modernas, explorando conceitos fundamentais de componentização, estilização e interação com o usuário.

Por meio deste projeto, os estudantes têm a oportunidade de aplicar na prática tecnologias amplamente utilizadas no mercado, desenvolvendo não apenas habilidades técnicas, mas também um olhar crítico e criativo para o design e a experiência do usuário.

Foram seguidas as referências desenvolvidas no [Figma](https://www.figma.com/design/mwZa1XHh9u0NYbhk3HImkn/Projeto---grupo-1?node-id=65-3&t=B73QbvKC5PCn9at8-1).

## Como rodar localmente

Primeiro instale as dependências do Node.js na raiz do projeto:

```bash
npm install
```
Com as dependências do Node instaladas, inicie o projeto com 
```bash
npm run dev
```

No terminal irá aparecer [http://localhost:3000](http://localhost:3000) (segurar "ctrl" e clicar no link irá abrir o projeto no seu navegador)
